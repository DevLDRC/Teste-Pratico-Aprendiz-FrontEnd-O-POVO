# Este projeto é parte de um teste de seleção de frontend onde nós vamos construir uma landing page para o filme da Barbie, como documentado no `./README.md`, e esse projeto tem regras e orientações de como seguir.

## Paleta de cores:

## Observações:
> Na seção de "Elenco", você vai inserir todos os atores do elenco, sendo que será carousel tanto no descktop como tablet e mobile. Para habilitar a função carousel, utilizamos está lib: https://owlcarousel2.github.io/OwlCarousel2/demos/demos.html mas fique a vontade para utilizar a que se sentir mais confortável.

> As imagens estão na pasta chamada "imgs" enumerados de 1 a 9.

> No botão "Ver mais" não precisa fazer nenhuma listagem, apenas a sua customização como sinalizada no layout.
